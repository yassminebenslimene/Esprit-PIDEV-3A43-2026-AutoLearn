📢 MESSAGE POUR L'ÉQUIPE - CONFIGURATION DES APIs
================================================

Salut l'équipe! 👋

J'ai pushé mon travail sur la branche Amira avec le module Événement complet.

⚠️ IMPORTANT: Les APIs ne fonctionneront PAS automatiquement chez vous!

🔑 POURQUOI?
============
Les clés API (SendGrid et OpenWeatherMap) ne sont PAS dans Git pour des raisons de sécurité.
Chaque personne doit configurer ses propres clés gratuites.

⚡ CONFIGURATION RAPIDE (5 minutes)
===================================

1️⃣ Après avoir pullé la branche Amira, créez votre fichier .env.local:
   
   copy .env.local.example .env.local

2️⃣ Obtenez une clé SendGrid (gratuit):
   - Créez compte: https://signup.sendgrid.com/
   - Settings → API Keys → Create API Key
   - Copiez la clé (commence par SG.)

3️⃣ Obtenez une clé OpenWeatherMap (gratuit):
   - Créez compte: https://home.openweathermap.org/users/sign_up
   - API keys → Copiez la clé
   - Attendez 10-15 min pour activation

4️⃣ Configurez .env.local avec vos clés:
   
   MAILER_DSN=sendgrid+api://SG.votre_cle@default
   WEATHER_API_KEY=votre_cle

5️⃣ Redémarrez le serveur Symfony

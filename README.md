# AutoLearn – E-Learning Platform

## Overview
This project was developed as part of the PIDEV – 3rd Year Engineering Program at Esprit School of Engineering (Academic Year 2025–2026).

AutoLearn is an online learning platform that allows students to access courses and manage learning content.

## Features
- Course management
- Chapter system
- Search
- Authentication

## Tech Stack

### Backend
- Symfony
- PHP
- MySQL

## Architecture
MVC Architecture using Symfony framework.

## Academic Context
Developed at Esprit School of Engineering – Tunisia  
PIDEV – 3A | 2025–2026  

## Getting Started
1. Clone the repository
2. Run composer install
3. Configure .env
4. Run symfony server:start

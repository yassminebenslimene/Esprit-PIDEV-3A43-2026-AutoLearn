Vérification dans Votre Application

  1. Connectez-vous en tant qu'étudiant
  2. Vérifiez que vous avez des notifications non lues en BDD:
    php bin/console doctrine:query:sql "SELECT COUNT(*) FROM notification WHERE user_id = 2 AND is_read = 0"
  3. Ouvrez la console du navigateur (F12)
  4. Vérifiez que l'API répond:
    fetch('/notifications/api/unread-count') .then(r => r.json()) .then(d => console.log('Notifications:', d))
  5. Le badge devrait apparaître automatiquement
💡 Astuce: Si le badge n'apparaît pas, vérifiez:
  • Que vous êtes connecté en tant qu'étudiant
  • Que vous avez des notifications non lues en base de données
  • Que le JavaScript s'exécute sans erreur (console F12)
  • Que le cache est vidé: php bin/console cache:clear

✨ EXEMPLES DE VALIDATIONS:

Quiz:
- Titre: 3-255 caractères, pas de caractères spéciaux
- Description: 10-2000 caractères
- État: actif/inactif/brouillon/archive uniquement

Question:
- Texte: 10-1000 caractères
- Points: 1-100 (entier positif)
- Quiz: obligatoire

Option:
- Texte: 1-255 caractères
- Correcte: oui/non obligatoire
- Question: obligatoire

🎯 CONCLUSION:

TOUTES les validations sont effectuées CÔTÉ SERVEUR uniquement.

AUCUNE validation HTML5 ou JavaScript n'est utilisée.

Les attributs HTML (maxlength, min, max) sont présents 
uniquement pour l'expérience utilisateur, mais NE SONT PAS 
utilisés pour la validation réelle.

La validation RÉELLE est faite par Symfony Validator côté 
serveur avec des contraintes définies dans les entités PHP.

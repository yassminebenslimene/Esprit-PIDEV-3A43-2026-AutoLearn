# ⚡ Guide Rapide - Assistant IA

## ✅ Statut: FONCTIONNEL

L'assistant IA est **100% opérationnel** et **intelligent**!

## 🎯 Comment l'Utiliser

1. **Ouvrez** http://127.0.0.1:8000/
2. **Connectez-vous**
3. **Cliquez** sur la bulle en bas à droite
4. **Posez** vos questions!

## 💬 Questions à Tester

### Salutations
```
"Bonjour"
"Hello"
"مرحبا"
```

### Cours
```
"Recommande-moi un cours"
"Aide-moi à choisir un cours"
"Quels cours pour débuter en Python?"
```

### Événements
```
"Événements cette semaine?"
"Quels événements à venir?"
```

### Statistiques
```
"Mes progrès?"
"Mon historique d'activités?"
```

### Aide
```
"Comment progresser rapidement?"
"Aide-moi"
```

## ✅ Ce Qui Fonctionne

- ✅ Réponses personnalisées (nom + niveau)
- ✅ Données réelles de la BD
- ✅ Cours avec détails complets
- ✅ Événements avec dates et places
- ✅ Statistiques utilisateur
- ✅ Multilingue (FR/EN/AR)

## 🚀 Pour Aller Plus Loin

**Installer Ollama (optionnel - 3 minutes):**

```bash
# 1. Installer
winget install Ollama.Ollama

# 2. Télécharger le modèle
ollama pull llama3.2:3b

# 3. Vérifier
ollama list
```

**Résultat:** Réponses encore plus naturelles et intelligentes!

## 🐛 Dépannage

### Erreur "Désolé, une erreur est survenue"
```bash
cd autolearn
php bin/console cache:clear
```

### Widget invisible
Vérifiez que vous êtes connecté.

### Réponses lentes
Normal si Ollama est installé (1-3 secondes).
Sans Ollama: instantané.
